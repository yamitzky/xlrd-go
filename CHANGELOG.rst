Changelog
=========

This changelog tracks the Go port of xlrd. For the original Python
history, see https://github.com/python-excel/xlrd.

0.1.0 (Unreleased)
------------------

- Initial Go port of the xlrd API for reading legacy ``.xls`` files.
- Core parsing, formatting, and formula evaluation support aligned with the
  migrated test suite.
- Hugo-based documentation scaffolding and migrated reference docs.
